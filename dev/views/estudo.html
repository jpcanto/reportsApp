Vamos falar sobre bind

Digamos que nós queremos encurtar um pouco nosso código. Temos 3 variáveis que buscam um Id como no exemplo:

let var1 = document.querySelector('#id1');

let var2 = document.querySelector('#id2');

let var3 = document.querySelector('#id3');

Agora, nós vamos tentar armazenar o document.querySelector em uma variável para encurtar nosso código e deixar mais elegante

let $ = document.querySelector;

let var1 = $('#id1');

let var2 = $('#id2');

let var3 = $('#id3');

(imitando o jquery rs) Muito melhor certo?

Claro, mas vai funcionar?

Não

Mas por que?

Vamos lá, o querySelector é um método que pertence a m objeto. Logo quando tentamos armazenar um método em uma variável, ele deixa de ser um método, se tornando uma função, perdendo assim sua associação a um objeto, no caso, o document.

Ok, como mantemos essa associação então? utilizando o Bind.

let $ = document.querySelector.bind(document);
